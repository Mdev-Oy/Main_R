-  Usage statistics
-  Achievements

## Technologies
- Firebase (BaaS)
- React Native
- CSS



## Prerequisites
- Node.js 
- Java
- Spring boot
- PostgresSQL


## Installation
1. Clone the repo
```sh
git clone https://github.com/Mdev-Oy/Main_R
```
2. Install dependencies
```sh
npm install
```
3. Put your firebase config in firebase.js
